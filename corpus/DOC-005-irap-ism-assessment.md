# IRAP Assessment Summary — Australian Government ISM (PROTECTED)

**Assessed entity:** Contoso Cloud Solutions Pty Ltd — Azure landing zone
reference architecture and managed services practice
**Assessment type:** IRAP (Infosec Registered Assessors Program) assessment
against the Australian Government **Information Security Manual (ISM)**,
June 2025 release
**Assessor:** Independent IRAP-endorsed assessor (report reference
IRAP-2025-0467, fictional)
**Classification level assessed:** PROTECTED

## Outcome summary

- Our Azure landing zone reference architecture was assessed as
  **effective** against applicable ISM controls at the PROTECTED level.
- 4 controls assessed as *alternate implementation*, each with documented
  compensating controls accepted by the assessor.
- No controls assessed as ineffective.

## Standing practices

- Landing zones deployed for government clients (including HealthGov,
  DOC-001) inherit this assessed architecture.
- Data sovereignty controls enforce deployment exclusively to Australian
  Azure regions (see data residency policy, DOC-006).
- Annual reassessment scheduled; next assessment window May 2026.

The full IRAP report is releasable to Commonwealth entities on request
through the appropriate security channels.
